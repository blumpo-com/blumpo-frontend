## Writing Your First Post

Creating a new post is easy with our CLI tool:

```bash
pnpm new:post "Your Awesome Title"
```

This will:

1. Create a new `.mdx` file with frontmatter
2. Set up a folder for images
3. Mark the post as a draft
4. Generate a URL-friendly slug

## Frontmatter Structure

Every post needs frontmatter metadata:

```yaml
---
title: Your Post Title
slug: your-post-title
date: 2025-11-02
tags: ["tag1", "tag2"]
excerpt: A brief description
draft: false
---
```
