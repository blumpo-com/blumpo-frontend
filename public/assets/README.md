# Assets Directory

This directory contains static assets for the dashboard UI.

## Structure

- `icons/` - Icon files (SVG, PNG) for sidebar navigation and UI elements
- `logo/` - Logo files for the Blumpo brand

## Usage

Icons and logo files should be placed in their respective folders. The components will reference these assets from the `/assets/` path.

### Example:
- Icon: `/assets/icons/wand.svg`
- Logo: `/assets/logo/blumpo.svg` or `/assets/logo/blumpo.png`
